## Getting Started

### Dependencies

* Rust Version 1.81.0 (2024-09-05) or higher
* Cargo 1.81.0 (2dbb1af80 2024-08-20)
* Command-line
* Git
* gh

### Installing

* Just clone using Git
```
git clone https://github.com/Xtrau505/Rust_Mem.git
```

### Executing program

* Enter the project directory
```
cd ./Rust_Mem
```
* Run using Cargo
```
cargo run --release
```
